= Commands
vib c - compile your project (you must in project with /cmd and bytes.yaml)
vib c {file}
vib t {file} {to programing lang} - translate your code to selected lang
vib t - you must in project
vib vm build - build from your project .object
vib vm compile {file .object} - compile to .vib-vm
vib run {your .vib-vm file}
vib install {install libs with bytes}
vib remove {remove libs with bytes}
vib build {build from your /build/build.hacker file}
vib help
vib docs
vib tutorials
